# ParaSnobFit Solver

Para-Snobfit minimizes a function over a box in $R^n$ (n = dimension of the problem).
   
## DOWNLOAD & INSTALL

1. Install the snobfit package: `python setup.py install`

1. Install the snobfit egg package: `python setup_egg.py install`
