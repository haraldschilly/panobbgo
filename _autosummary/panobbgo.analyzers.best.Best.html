panobbgo.analyzers.best.Best¶

class panobbgo.analyzers.best.Best(strategy)[source]¶

Listens on all results, does accounting for “best” results, manages a pareto front of good points and emits the following events:

  • new_best: when a new “best” point,

  • new_min: a point with smallest objective function value,

  • new_cv: one with a smaller cv, or

  • new_pareto: when a best point on the pareto front of the objective value and constraint violation

    has been found.

Note

Pareto: Once the constraint violation is zero, only the objective function value is considered and future results with a positive constraint violation are ignored.

The best point (pareto) is available via the best attribute. There is also cv, min and pareto.
