panobbgo.strategies.ucb.StrategyUCB¶

class panobbgo.strategies.ucb.StrategyUCB(problem, **kwargs)[source]¶

This strategy uses the Upper Confidence Bound (UCB1) algorithm to select heuristics. It balances exploration (trying less used heuristics) and exploitation (using heuristics that have performed well).

The Multi-Armed Bandit formulation here is: - Arms: Heuristics - Pull: Generating points from a heuristic - Reward: 0 if point doesn’t improve best; >0 if it improves best.

Reward calculation: R(x) = 1 - e^{-(improvement)}.

  • Value Q_t(a): Average reward per point generated.
